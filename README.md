This is my pet-project for a very cool handmade retro futuristic furniture store. I realize that using `React` for this kind of website is an overkill and any CMS would be more than enough. This is still a training project for me with a practical use for the seller - PlyPie. Thanks to the owner for this opportunity ‚ù§
